# Pyaint-Shape-Module
An open source paint program using Python and Pygame. Features will be added by the community as time goes on.

Features:

1)Free Drawing 
2)Shape Drawing 
3)Filling a Shape With Color 
4)Unfilling A Shape 
4)Shape Dragging 
5)Shape Resizing 
6)Shape Copy Pasting 
7)Changing Thickness Of A Shape
